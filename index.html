Questions & Answers



Q1: Mention three differences between Web1.0, Web2.0, Web3.0

Difference Web 1.0 Web 2.0 Web 3.0
User Abilities Read-Only Read-Write Read-Write-Execute
User Interaction The pages were static and were used as portals with links, users can navigate through the links to find information, and read the pages Users can search for information, and publish content that other users can consume, share and comment on Personal and portable, applications are using AI technique (natural language) to personalize user experience.
Primary Streaming Media Home Pages Wikipedia, Blogs, Facebook, YouTube Live-Streaming, Waze
Communication Message Boards, personal web sites Community portals, social networks Semantic social networks, semantic forums


Q2: Explain the differences by using use case

The user will search for the term "python".
Difference Web 1.0 Web 2.0 Web 3.0
User Interaction The user has to do the search manualy by going through hyperlinks. The user will enter the term "python" in a search engine, he will get links to information about python (most likely the animal) and possibly links for photos and videos, results might not be relevant The user will enter the term "python" in a search engine, and will get relevat and personilized information like if the user is intrested in programming the result might include recommended Courses for learning python, or information relevant to his location like meet-ups
Primary Streaming Media The pages that will be provided are home pages with the subject of python The pages that will be provided are python's Wikipedia page and photos and videos about python, the user can comment and ask questions The pages that will be provided are youtube's videos about python, and related subjects like programming, data analysis using python, and questions that people asked about python
Communication The user can post to his personal site his interest with python The user can share and comment via social platforms his interest in python The user can participate in live streams, and live online courses related to python
